ㅇ주관: 한국데이터산업진흥원(Kdata)
ㅇ내용: 데이터베이스의 기본 개념과 SQL 문법, 데이터 모델링 등을 다루며, SQL 개발자로서의 기초 역량을 평가합니다.
ㅇ준비: 공식 교재와 문제집을 통해 준비할 수 있으며, 온라인 강의를 병행하면 좋습니다.
